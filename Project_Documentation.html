Smart Kiryana (Smart Store)

Complete Inventory Management Solution

1. Project Overview

Smart Kiryana is a comprehensive inventory management system designed for small business owners and retail stores. It provides a multi-platform solution (Web and Mobile) to track products, manage multiple shop owners, and ensure data safety through cloud backups.

2. Key Features

🔐 User Authentication & Security

  • Multi-Provider Login Supports traditional Email/Password registration and Social Login (Google).
  • Secure Storage Passwords are encrypted using modern hashing algorithms (PBKDF2) for maximum safety.
  • Dual-Mode Auth Uses secure sessions for Web and Header-based (X-User-Id) authentication for Mobile.
  • Auto-Logout Includes a 2-hour inactivity timer to protect sensitive store data from unauthorized access.

📦 Inventory & Shop Management

  • Multi-Shop Support A single user can manage multiple business owners or shop locations independently.
  • Comprehensive Product Tracking Tracks names, categories, units, sale prices, cost prices, and stock levels.
  • Real-time Dashboard Provides instant summaries of total products, inventory value, and stock status.

☁️ Cloud Integration & Experience

  • Google Drive Backup Automated synchronization of inventory data to a dedicated Google Drive folder.
  • Dynamic Theming Automatically detects and applies the user's system theme (Light or Dark mode).
  • Mobile Responsive Optimized interface that works seamlessly across desktops, tablets, and smartphones.

3. How the App Works (Workflow)

Step 1: Onboarding

The user registers or logs in. The system identifies the user and loads their specific settings, including their preferred theme.

Step 2: Business Setup

The user adds their shop details under the "Owners" section. Multiple shops can be created for different branches.

Step 3: Inventory Entry

Products are added with cost and sale prices. The system automatically calculates the total business value and inventory health.

Step 4: Cloud Sync

Whenever data changes, it can be synced to Google Drive, ensuring that the business record is never lost.

Step 5: Daily Monitoring

Owners use the real-time search and stock indicators to manage daily sales and reordering tasks efficiently.
